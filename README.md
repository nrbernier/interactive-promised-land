# Interactive Promised Land

An interactive map exploring the history of Israel/Palestine, from roughly 1000 BCE to today.

Drag the time slider to see which nation, kingdom, or political entity controlled each part of the region at any point in history. Click a territory for a summary, illustration, and links to further reading (Wikipedia, etc). Key historical events are marked above the timeline — hover or click for details.

## Usage

Open `index.html` in a browser, or serve the directory locally:

```
python3 -m http.server
```

Then visit `http://localhost:8000`.
