# Idea

I would like an interactive experience to better understand the history of
Israel/Palestine.

It should be a map of the region, showing neighbooring countries. There should
be a slider that allows to select the time (from around -1000 CE to now). Then
moving the slider the map will show which nation/country/political entity fills
each area. Each element should be clickable to show a summary with illustration
and links to sources (wikipedia, etc).

There should be also events shown above the time slider at key historical
moments. Each event should be hoverable and also clickable with additional
information as above.
